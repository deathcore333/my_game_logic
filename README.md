# my_game_logic
Contains a number of logic for different games. All the logic will be integrated to play in the terminal.
Game Logic Repository
This repository contains logic for different games that can be played in the terminal. These games have been implemented using Python 3.

Installation
Clone this repository to your local machine using git clone.
Navigate to the directory containing the game you want to play.
Run the Python file using python <filename>.py.
List of Games
1. Truth or Dare
This is a classic party game where players take turns choosing between answering a truth question or performing a dare. The game can be played with any number of players, and the questions or dares can be customized to suit the group.

2. Rock Paper Scissors
This is a classic two-player game where players choose between rock, paper, and scissors. Rock beats scissors, scissors beat paper, and paper beats rock. The player who wins best out of three rounds wins the game.
